# Tổng quan

## Thông tin kết nối

Chuẩn bị:

* Đăng ký tài khoản [https://sso.omicall.io/](https://sso.omicrm.com/)
* Truy cập **Cấu hình > Doanh nghiệp >> Tích hợp > WEBHOOK**
* Cung cấp URL để lấy thông tin OmiCall gửi (url bắt buộc phải **https**)
