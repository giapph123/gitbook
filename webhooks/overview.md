# Tổng quan

## Thông tin kết nối

Chuẩn bị:

* Đăng ký tài khoản [https://sso.omicall.com/](https://sso.omicrm.com/)
* Truy cập **Tài khoản &gt; Cấu hình &gt; Quản lý API &gt; WEBHOOK**
* Cung cấp URL để lấy thông tin OmiCall gửi \(url bắt buộc phải **https**\)
